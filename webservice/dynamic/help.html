/html

For api-like html requests

On every page that is served, each tag containing a piece of data, eg the user's rating will have an attribute name with the value of the name of the data that tag contains.
For example: 

View count: 3
This way, to find any data from any page, you can do a name= query selection. or regex search

/by-id/[id]

Serves an html page containing information about the item id id

If id is not provided, a list of items in a table will be served

/search

the parameter query is required for a search query.

returns a table containing all search results
